# Contributing to Enhancing ClinicalBERT for ICD Coding

Thanks for your interest in contributing! Please follow these guidelines to make collaboration smooth.

## Reporting Issues
- Use the repository's Issues to report bugs or suggest enhancements.
- Provide a clear title, steps to reproduce, and relevant environment details.

## Proposing Changes
- Open an Issue first for non-trivial changes to discuss design.
- Create a feature branch from `main` named `feat/short-description` or `fix/short-description`.

## Development Setup
1. Create a virtual environment: `python -m venv .venv`
2. Activate it and install dependencies: `pip install -r requirements.txt`

## Coding Style
- Follow existing code style. Prefer readable, well-documented changes.
- Optionally run formatters like `black` if available in your environment.

## Tests
- Add tests when adding functionality. Describe how to run them in the PR.

## Pull Requests
- Open a PR against `main` with a clear description and link to the related Issue.
- Keep changes small and focused. Squash or rebase as needed.

## License
By contributing, you agree that your contributions will be licensed under the project's existing license.

Thank you — contributions help improve reproducibility and usefulness for the community.
